# Mohamed Mubeen A S — Portfolio

A modern, responsive portfolio website for **Mohamed Mubeen A S**, Full Stack Developer based in Dubai, UAE. Built with React, TypeScript, and Tailwind CSS.

**Live:** [TAP LINK](https://portfolio-mxbn.vercel.app/)



## ✨ Features

- **Hero** — Animated intro with role rotation and interactive background
- **About** — Short bio and professional focus
- **Skills** — Tech stack and competencies
- **Experience** — Work history and roles
- **Projects** — Showcase of projects
- **Responsive** — Mobile-first, works on all screen sizes
- **Dark theme** — Clean dark UI with smooth animations

## 🛠️ Tech Stack

React · TypeScript · Vite · Tailwind CSS · shadcn/ui

## 📁 Project Structure

```
src/
├── app/           # App shell, routing, providers
├── features/      # Page sections (Hero, About, Skills, Experience, Projects, Contact)
├── pages/         # Route pages (Index, NotFound)
└── shared/        # Reusable components, hooks, UI, utils
```

## 🚀 Install & Run

**Prerequisites:** Node.js 18+ and npm (or pnpm/yarn)

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/portfolio_mubeen.git
cd portfolio_mubeen

# Install dependencies
npm install

# Run the app
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
